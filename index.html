Welcome on Mo'Blocks Official Wiki

Mo'Blocks is a simple mod that adds random decorative and useful blocks.

Currently being worked on by KiroTheBlueFox, Mo'Blocks adds various blocks to the game, such as ramps, archs or even pillars ! This mod is currently in 1.14.4 and it needs forge to work.

v0.7 is finally here !


Be Careful :

Optifine doesn't like Mo'Blocks, as it adds thousands and thousands of models and optifine doesn't appreciate that

v0.7
Added :
  • Posts : Available in all block types
  • Furnitures :
    • Tables : Automatically connecting to each others
    • Chairs : Available in all 16 directions, can be sat on
    • Coffee Tables : Automatically connecting to each others
    • Sofas : Working like stairs, can be sat on
    • Round Carpets : Automatically connecting to each others
    • Drawers : Simples & Doubles, have respectively 16 and 32 slots
    • Crates : Has 64 slots but no GUI
    • Shelves : Has 4 slots but no GUI
    • Bookshelves : Has 16 slots but can contain only book items (book, written books and enchanted books)
    • Lamps : Can be turned on/off by hand or with redstone
    • Lamp Shades : exist in 10 shapes as well as in all 16 dye colors
    • Tall Fences
    • Tall Fence Gates : Work like doors
    • Candles :
      • Can be colored using a Custom Color Picker
      • NBT Tags :
        • color [integer] : defines the colors
    • Colorable Flower Pots :
      • Can be colored using a Custom Color Picker
      • NBT Tags :
        • color [integer] : defines the colors
        • flower :
          • id [registry name] : defines the flower
          • count : how much flower [max=1]
  • Special items and blocks :
    • Block Colorer : use in crafting table with a block to make it colorable using the Custom Color Picker (works with planks, terracotta, concrete, wool, bricks, stone bricks, stone, glass, glass panes & cobblestone)
    • Custom Color Picker :
      • Use in the crafting table like leather armors to color it
      • Sneak + Right click on an already colored block to pick its color
      • Can be reset using a cauldron with water
      • Can be given using the tab
      • NBT Tags :
        • display :
          • color [integer] : defines the color
    • Colorable Blocks : Can be colored using a Custom Color Picker on them
    • Colorable Furnitures : Furnitures that can be colored : Tables, Carpets, Round Carpets, Sofas, Chairs, Plank Coffee Tables, Terracotta Coffee Tables
Changed :
  • Crafting recipes :
    • Most crafting recipes were moved to the stone cutter, so there is no more glitches and it is way easier to craft
  • Code :
    • Rewrote a part of the code of certain blocks to make them way less glitched
Removed :
  • Hammer
  • Disco balls
  • Illusion block

I'm new to modding so maybe there will be some glitches with certain blocks. I really hope this mod will be appreciated ! Any feedback, critic, custom block requests/suggestions or anything else is welcomed !
